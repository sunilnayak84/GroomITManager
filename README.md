# Pet Grooming Management System

A comprehensive management system for pet grooming businesses that streamlines operational workflows and enhances customer service.

## Features

- Appointment scheduling with conflict detection
- Customer and pet management
- Service package management
- Inventory tracking
- Role-based access control (RBAC)
- Staff management
- Real-time updates

## Tech Stack

- TypeScript with strict type checking
- React with Hooks
- Firebase Authentication
- Express.js backend
- PostgreSQL database with Drizzle ORM
- Tailwind CSS for styling
- Full Calendar for appointment management

## Development

To run the development server:

```bash
npm run dev
```

## Environment Setup

The application requires the following environment variables:

- `DATABASE_URL`: PostgreSQL database connection string
- Firebase configuration (various keys)
- `PORT`: Server port (defaults to 3000)
